# Project Title: Siem and Automated ticketing system integration handson practice

## 📌 Overview
This repository is a documentation of Wazuh and OsTicket hands-on practice homelab in a virtualize environment using Hyper-V. It demonstrates the different features and functions and also how to navigate each of the software, how to install agent on each devices(endpoint) in a network to monitor their logs using wazuh manager, and how to use OsTicket including creating tickets, assigning an agent to the ticket, lastly is an automated ticketing system by integrating wazuh and osticket to automatically create a ticket.

---

## 🎯 Objectives
- Gain a hands-on experience in using both Wazuh and Osticket.
- Learn how to navigate Wazuh Dashboard and monitor logs on each endpoint.
- Learn how to create ticket using Osticket and how to assign an agent to a ticket.
- Learn how does the SIEM and Ticketing system actually works together in a real enterprise environment.
- Learn how to integrate Wazuh and Osticket for automated ticketing system

---

## 🖥️ Environment
- Ubuntu Server (SIEM and Ticketing System)
- Wazuh Manager and Dashboard
- Wazuh Agents (endpoints)
- Osticket
- LAMP Stack
- Hyper-V
