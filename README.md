# Project Title: Siem and Automated ticketing system integration handson practice

## 📌 Overview
This repository is a documentation of Wazuh hands-on practice homelab in a virtualize environment using Hyper-V and also a hands-on practice using the ServiceNow platform Incident Management tool. It demonstrates the different features and functions and also how to navigate each of the software, how to install agent on each devices(endpoint) in a network to monitor their logs using wazuh manager, and how to use ServiceNow incident management tool including Creating an Incident, User and also assigning a User or a Group into an incident. It also demonstrate an automated incident creation by integrating Wazuh and ServiceNow Incident Management Tool.

---

## 🎯 Objectives
- Gain a hands-on experience in using both Wazuh and ServiceNow.
- Learn how to navigate Wazuh Dashboard and monitor logs on each endpoint.
- Learn how to use different features and functionalities of ServiceNow Incident Management Tool including.
- Learn how does the SIEM and Incident Management System actually works together in a real enterprise environment.
- Learn how to integrate Wazuh and ServiceNow for automated ticketing system

---

## 🖥️ Environment
- Ubuntu Server (SIEM and Ticketing System)
- Wazuh Manager and Dashboard
- Wazuh Agents (endpoints)
- ServiceNow (Incident Management Tool)
- Hyper-V
